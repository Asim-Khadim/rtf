Constructs the object

__construct(string $source = null)

If $source string is passed and it starts with a "{" character), I'll attempt to read() the source. If $source string is passed and it does not start with a "{" character, I'll assume it's a file name and attempt to load it. If $source is not passed, I'll instantiate the document's root group.

since 0.1.0

Arguments

$source

string

a file name (must not start with "{" character) or
